JavaScript Functions: Algorithms

functions documented

convertCtoF - convert degreees in celsius to fahrenheit.

reverseString - reverse a string

factorializeNumber - factorialize a number.

findLongestWordLength - find the length of the longest word in a string.

largestOfFour - return an array consisting of the largest number from each sub-array. example : [[4, 5, 1, 3], [13, 27, 18, 26], [32, 35, 37, 39], [1000, 1001, 857, 1]]

confirmEndingLetter - Check if a string ends with the letter inputed

confirmEndingTarget - Check if a string ends with the contents inputed

repeatStringNumTimes - repeat a string as many times as inputed

truncateString - Truncate a string if it is longer than the length inputed.

findersKeepers - input an array and return the first number that is even.

booHoo - checks if the input is a boolean value.

titleCase - Return the inputed string with the first letter of each word capitalized. Make sure the rest of the word is in lower case

sliceAndSplice - copy each element of the first array into the second array, in order, using the input of "index" as the placement at which to add

falsyBouncer - Remove all falsy values from an array.

whereDoIBelong - Return the lowest index at which a value (second argument) should be inserted into an array (first argument) once it has been sorted. The returned value should be a number.

mutations - Return true if the string in the first element of the array contains all of the letters of the string in the second element of the array.

chunkyMonkey - Write a function that splits an array into groups the length of size, and returns them as a two-dimensional array.
